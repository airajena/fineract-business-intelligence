README 

The Business Intelligence Repo was started May 2026 as part of the GSOC program 

It provides tooling to take data from Fineract on an operational basis and run it through various analytics for business intelligence. 

A key part of the concept is to be "downstream" and separate from Fineract.   

The stack will use other open source projects and be highly performant for a range of deployments.
